AI Models

The three AI tiers of Pollo Assistance.

🆕 New in v2: Each model (Palos, Luces, Summum) now acts as an orchestrator brain that decides on every message which AI from the pool to use (Llama, Qwen, GPT-OSS, Kimi K2…) based on the task: fast chat, code, deep reasoning, multilingual, etc. It also decides when to search the web, when to draw an interactive diagram, when to activate chess mode, and when to think before answering (think mode) — all automatically.

  • Smart search: when the brain decides it needs updated info, it performs a web search and shows the Sources button with the pages consulted.
  • Inline diagrams: complex explanations are rendered as interactive mind maps inside the chat.
  • Think mode: for difficult tasks, the Luces and Summum models show a collapsible panel with their internal reasoning before the response.
  • Integrated chess: the brain decides when to open a playable board inside the chat.
  • Chat memory: the AI can remember previous conversations to provide relevant context (configurable in Settings).

How the models work

Pollo Assistance offers three artificial intelligence models with different capability levels. Each model is optimized for a type of use, from quick responses to complex and deep analysis.

The models are based on the Meta LLaMA family, run through the Groq infrastructure, which provides very low response times thanks to its specialized hardware (LPU).

Important: Once you send the first message in a chat, the model is locked for that conversation. If you want to change the model, you need to create a new chat.

Pollo Palos v1.0

Base modelLLaMA 3.1 8B (8 mil millones de parametros)
AvailabilityFree
Max tokens500 por respuesta
StreamingNo
Best forQuick responses and simple questions

Pollo Palos is the lightest and fastest model. It uses a model with 8 billion parameters, which allows it to give near-instant responses. It is ideal for short questions, quick translations, simple calculations or queries that do not require deep analysis.

Having fewer parameters, its responses are more concise and it may struggle with very complex tasks or those requiring long reasoning. It is the perfect model when you need something fast and to the point.

Pollo Luces v1.0

Base modelLLaMA 3.3 70B (70 mil millones de parametros)
AvailabilityFree
Max tokens1000 por respuesta
StreamingYes (progressive response)
Best forDaily use, normal conversations

Pollo Luces is the recommended model for daily use. With 70 billion parameters, it offers more complete responses, better reasoning and greater ability to understand context and nuance. It also supports streaming, meaning the response appears progressively on screen (word by word) instead of waiting for it all to generate.

It is the best option for normal conversations, task help, text writing, detailed explanations and most everyday uses.

Pollo Summum v1.0

Base modelLLaMA 3.3 70B (70 mil millones de parametros)
AvailabilityPremium only
Max tokens2000 por respuesta
StreamingYes (progressive response)
Best forDeep analysis, complex tasks

Pollo Summum is the most powerful model on the platform, exclusive to premium users. Although it uses the same 70B architecture as Luces, Summum has double the allowed tokens (2000 vs 1000), which allows it to give much longer and more detailed responses.

It is the ideal model for tasks requiring exhaustive analysis: long writing, complex problem solving, extensive code generation, chess game analysis and any situation where depth and detail are needed.

AI Vision

In addition to the text models, Pollo Assistance includes an artificial vision model for image analysis:

ModelLLaMA 4 Scout (Groq Vision)
FunctionImage analysis and description
AvailabilityFree and Premium

This model activates automatically when the user uploads an image to the chat. The AI analyzes the visual content and can describe the image, answer questions about it or, in the case of chess board images, start a game analysis.

Which one to choose

Palos if you need something fast and simple. Luces for daily use (the recommended option). Summum when you need depth and detail (requires premium).
