What is Pollo Assistance

Pollo Assistance is an AI ecosystem that brings together on a single platform all the AI tools a user might need: smart chat, image generation, visual analysis, web search, voice input and output, chess game analysis, interactive diagrams and more.

What sets Pollo Assistance apart from other AI assistants is its one-time payment model: for $25 (no monthly subscriptions) all features are unlocked for life. There is also a free plan with 20 messages every 6 hours to try the platform.

Additionally, Pollo Assistance has its own personality: it presents itself as "the supreme chicken", with a friendly, approachable and humorous tone without losing usefulness. It uses expressions like "Cocorocó!" and chicken emojis in its interactions.

Main features

AI Chat

The core of Pollo Assistance is a conversational AI chat. Users can ask questions, request help, generate content and hold natural conversations. The AI has context memory: it remembers what has been discussed in the current conversation and can search previous chats to give more relevant responses.

Image generation

Directly from the chat, users can ask for an image to be generated by describing what they want. The AI interprets the description and generates the image instantly. No need to open another app or switch windows.

Image analysis

Users can upload images for the AI to analyze. Pollo Assistance uses Groq Vision (based on LLaMA 4 Scout) to describe, interpret and answer questions about any image sent to it.

Web search

When a question requires updated information, the AI automatically searches the internet and presents the results with original sources. The user can see the consulted URLs alongside the response.

Voice

Pollo Assistance supports voice input (voice recognition to dictate messages) and voice output (the AI reads its responses aloud). This allows using the platform in hands-free mode.

Chess analysis

A unique feature: users can upload an image of a chess board or paste a game in PGN format, and the AI analyzes every move, identifies the opening, evaluates plays and offers an interactive board to navigate the game.

Interactive diagrams

The AI can generate concept maps and visual diagrams from any text or topic. The diagrams are interactive, with nodes that can be expanded and collapsed to explore information at different levels of depth.

Chat memory

Pollo Assistance can search through the user's previous conversations to provide context for new questions. This feature can be enabled in settings and helps the AI give more personalized responses over time.

Ecosystem modules

In addition to the main chat, Pollo Assistance includes specialized modules:

  • Pollo STUDY — Academic assistant for students.
  • Pollo CODE — AI-powered programming environment.
  • Pollo FILES — Document and PDF analysis.
  • Pollo GO — Quick utilities and smart navigation.

Available platforms

Pollo Assistance is available on two platforms:

  • Web: Accessible from any modern browser at pollo-assistance.web.app. No installation required.
  • Android: Native app that loads the web platform in a mobile-optimized environment, with system feature integration (notifications, sharing, etc.).

Technology

Pollo Assistance uses the following technologies:

  • AI Models: Meta's LLaMA 3.x family, run through Groq API for low latency.
  • AI Vision: LLaMA 4 Scout via Groq Vision for image analysis.
  • Backend: Firebase (Firestore for database, Authentication for accounts, Hosting for web).
  • Frontend: HTML, CSS and JavaScript with no external frameworks.

Integration with AuraEON

Pollo Assistance works as the central account and AI system for AuraEON, an educational project for schools. Students using AuraEON access the AI through Pollo Assistance.
