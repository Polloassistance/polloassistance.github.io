What is Pollo Assistance

Pollo Assistance is an AI ecosystem that brings together on a single platform all the AI tools a user might need: smart chat, image generation, visual analysis, web search, voice input and output, chess game analysis, interactive diagrams and more.

What sets Pollo Assistance apart from other AI assistants is its one-time payment model: for $25 (no monthly subscriptions) all features are unlocked for life. There is also a free plan with 20 messages every 6 hours to try the platform.

Additionally, Pollo Assistance has its own personality: it presents itself as "the supreme chicken", with a friendly, approachable and humorous tone without losing usefulness. It uses expressions like "Cocorocó!" and chicken emojis in its interactions.

Main features

AI Chat

El nucleo de Pollo Assistance es un chat conversacional con inteligencia artificial. Los usuarios pueden hacer preguntas, pedir ayuda, generar contenido y mantener conversaciones naturales. La IA tiene memoria de contexto: recuerda lo que se ha hablado en la conversacion actual y puede buscar en chats anteriores para dar respuestas mas relevantes.

Image generation

Directamente desde el chat, los usuarios pueden pedir que se genere una imagen describiendo lo que quieren. La IA interpreta la descripcion y genera la imagen al instante. No hace falta abrir otra aplicacion ni cambiar de ventana.

Image analysis

Los usuarios pueden subir imagenes para que la IA las analice. Pollo Assistance usa Groq Vision (basado en LLaMA 4 Scout) para describir, interpretar y responder preguntas sobre cualquier imagen que se le envie.

Web search

Cuando una pregunta requiere informacion actualizada, la IA busca automaticamente en internet y presenta los resultados con las fuentes originales. El usuario puede ver las URLs consultadas junto a la respuesta.

Voice

Pollo Assistance soporta entrada por voz (reconocimiento de voz para dictar mensajes) y salida por voz (la IA lee sus respuestas en voz alta). Esto permite usar la plataforma en modo manos libres.

Chess analysis

Una funcion unica: los usuarios pueden subir una imagen de un tablero de ajedrez o pegar una partida en formato PGN, y la IA analiza cada movimiento, identifica la apertura, evalua jugadas y ofrece un tablero interactivo para navegar la partida.

Interactive diagrams

La IA puede generar mapas conceptuales y esquemas visuales a partir de cualquier texto o tema. Los esquemas son interactivos, con nodos que se pueden expandir y colapsar para explorar la informacion a diferentes niveles de profundidad.

Chat memory

Pollo Assistance puede buscar en conversaciones anteriores del usuario para aportar contexto a nuevas preguntas. Esta funcion es activable desde los ajustes y ayuda a que la IA de respuestas mas personalizadas con el tiempo.

Ecosystem modules

Ademas del chat principal, Pollo Assistance incluye modulos especializados:

  • Pollo STUDY — Asistente academico para estudiantes.
  • Pollo CODE — Entorno de programacion con IA.
  • Pollo FILES — Analisis de documentos y PDFs.
  • Pollo GO — Utilidades rapidas y navegacion inteligente.

Available platforms

Pollo Assistance esta disponible en dos plataformas:

  • Web: Accesible desde cualquier navegador moderno en pollo-assistance.web.app. No requiere instalacion.
  • Android: Aplicacion nativa que carga la plataforma web en un entorno optimizado para movil, con integracion de funciones del sistema (notificaciones, compartir, etc.).

Technology

Pollo Assistance utiliza las siguientes tecnologias:

  • Modelos de IA: Familia LLaMA 3.x de Meta, ejecutados a traves de Groq API para baja latencia.
  • Vision por IA: LLaMA 4 Scout via Groq Vision para analisis de imagenes.
  • Backend: Firebase (Firestore para base de datos, Authentication para cuentas, Hosting para la web).
  • Frontend: HTML, CSS y JavaScript sin frameworks externos.

Integration with AuraEON

Pollo Assistance funciona como el sistema central de cuentas e inteligencia artificial de AuraEON, un proyecto educativo para centros escolares. Los estudiantes que usan AuraEON acceden a la IA a traves de Pollo Assistance.
